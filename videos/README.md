# Project Overview: LCD Display Demos

The following videos were recorded by capturing a **7-inch LCD screen** in a darkened room to ensure optimal visibility of the display.

---

### Video Descriptions

| File Name | Description |
| :--- | :--- |
| **demo_auto.mp4** | Demonstrates the graphical capabilities of **MicroPython** using the **LVGL** library. |
| **demo_car.mp4** | Shows a simulation of an **electric car dashboard** (instrument cluster). |

---

### Technical Details
* **Hardware:** 7-inch LCD Panel
* **Language:** MicroPython
* **Graphics Library:** LVGL (Light and Versatile Graphics Library)
* **Environment:** Low-light recording for high-contrast visualization
